# Financial Analysis & Automation 📊

## Project: Loan Interest Calculator
This Python tool automates the calculation of loan interest based on principal amounts and annual rates. It is designed to replace manual spreadsheet entries with a scalable, code-driven solution for financial modeling.

### Key Features
* **Dynamic Inputs:** Easily adjust principal, rates, and timeframes.
* **Automated Logic:** Uses Python 3 to ensure consistent, error-free results.
* **Professional Workflow:** Developed in Google Colab and managed via GitHub.

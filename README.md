### Key Features
* **Dynamic Inputs:** Easily adjust principal, rates, and timeframes.
* **Automated Logic:** Uses Python 3 to ensure consistent, error-free results.
* **Professional Workflow:** Developed in Google Colab and managed via GitHub.
